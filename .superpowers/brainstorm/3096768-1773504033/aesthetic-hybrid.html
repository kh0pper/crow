Hybrid: Clean Sans + Editorial Details

Sans-serif message text for clean readability. Monospace for metadata (timestamps, model names, file sizes). Subtle borders for structure. Rounded squircle avatars. Status badges. File attachments with preview. Thread replies with left-border indicator. Double-check delivery status.

Key Design Decisions in This Mockup

Typography
System sans-serif for messages (optimized for reading). JetBrains Mono for timestamps, model info, file sizes. Fraunces serif stays on panel titles only.
Color System
Indigo (#6366f1) for sent messages + accents. Green for online status. Gold for unread badges. Red for destructive actions. Muted grays for metadata.
Avatars
Squircle (rounded-square) instead of circles — more modern, like iOS/Telegram. Gradient for AI contacts. Solid colors for people. Active border glow on selected.
Info Panel
E2E encryption status + safety number. Shared files list. Shared items (memories, projects). Quick actions (share, block). Collapsible on mobile.
