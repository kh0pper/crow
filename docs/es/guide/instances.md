# Encadenamiento Multi-Instancia

Conecta múltiples instalaciones de Crow en diferentes máquinas o directorios. Las instancias sincronizan memorias, contexto y contactos automáticamente vía Hypercore P2P.

Ver la [versión en inglés](../../guide/instances) para la documentación completa.
